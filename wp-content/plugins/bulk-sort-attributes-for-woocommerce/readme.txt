Bulk sort WooCommerce attributes when they are too numerous for custom sorting by hand.

== Description ==

Bulk Sort Attributes for WooCommerce is a WooCommerce extension that allows you to bulk 
order WooCommerce attributes by ID, Name, or Name (Numeric). When there are 
hundreds or even dozens of items in an attribute it becomes tedious and time 
consuming to sort by hand. This plugin makes it an easy task!


== Installation ==

1. Unzip the plugin file and extract the folder.
2. Upload the bulk-sort-attributes-for-woocommerce folder to the /wp-content/plugins/ directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
